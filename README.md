# wn_blocks
###### The module adds three blocks to the homepage of the shop, based on the PrestaShop engine. It uses the displayHome hook.

------------
# Requirements
PHP 7.4.26+
- PrestaShop 1.7.8.7 (recommended)
- Composer 2.0.23
- Node 16.16.0
# Install
Run in the console
1. `composer install`

2. `yarn`
